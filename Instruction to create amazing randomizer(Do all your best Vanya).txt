
Формат данных - массив объектов. Пример одного из объектов массива

{
nation:"Indonesia",
image:"./img/nations/indonesia.png",
}

Игрок, зайдя на страницу, выбирает колличество игроков, которые будут участвовать в розыгрыше наций. 
Под выбранное колличество людей, рендерится список игроков, и так же в этом списке идёт список наций (ИВАН, БЕЗ ТАБЛИЦ!!).
На странице так же будет имется кнопка "пересоздать"


Фичи и приколы:

1)Сделать сборку через Веб пак и для шаблонов использовать .HBS файлы.
2)Добавить прелоадер который будет отображаться ровно 2 секунды.
3)По возможности сохранять данные в локальном хранилище.
4)Добавить пару интересных плагинов, "оживить" страничку.
5)По желанию добавить музыкальное сопровождение.
6)Сделать собственный, красивый дизайн схожий с главным меню в самой игре.
